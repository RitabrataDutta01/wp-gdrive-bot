# WhatsApp Google Drive Auto Folder Sender

A simple Python project that automatically creates a Google Drive folder for each day and sends the folder link via WhatsApp. Designed to streamline daily file organization and sharing.

---

## Features

- Creates a new Google Drive folder with the current date as its name.
- Sends the folder link instantly to a specified WhatsApp contact using `pywhatkit`.
- Stores WhatsApp Chromium session locally to keep you logged in across runs.
- Fully configurable via a `.env` file.
